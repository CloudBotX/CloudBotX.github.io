Java è ideale per applicazioni robuste e scalabili. Lo utilizzo in progetti server-side e per creare programmi con architetture più complesse.

JavaScript

JavaScript è il cuore dell’interattività web. Lo uso per animazioni, effetti dinamici e per rendere i miei siti moderni e responsive.

HTML

HTML è la struttura base dei miei siti. Lo combino con CSS e JS per creare interfacce eleganti, intuitive e compatibili con tutti i dispositivi.

CSS
